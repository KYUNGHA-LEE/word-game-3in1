# 단어 게임 — 초성 · 자모 · 한붓

CSV로 단어 문제를 불러와 초성·자모·한붓 세 가지 방식으로 풀 수 있는 단일 HTML 단어 게임입니다. 별도의 설치나 서버 없이 브라우저에서 바로 실행되며, 수업 현장에서 원하는 문제로 즉시 교체해 사용할 수 있습니다.

## 주요 기능

- **세 가지 게임 방식**
  - 초성: 정답의 초성만 보여 주고 단어를 입력합니다.
  - 자모: 흩어진 자음·모음을 보고 단어를 입력하며, 정답을 맞히면 자모가 모여 단어로 완성됩니다.
  - 한붓: 5×5 글자판에서 인접한 칸을 순서대로 이어 단어를 완성합니다.
- **CSV로 문제 불러오기**: 엑셀·구글시트에서 내보낸 CSV 파일을 불러오거나 화면으로 끌어다 놓으면 됩니다. 첫 행 머리글은 자동으로 건너뜁니다.
- **샘플 체험**: 파일 없이도 내장된 예시 문제로 곧바로 사용해 볼 수 있습니다.
- **단계별 힌트**: 초성·자모는 일정 횟수 이상 틀리면 글자를 하나씩 공개하고, 한붓은 첫 글자 위치 표시와 정답 경로 안내를 제공합니다.
- **진행 표시와 기록 시간**: 문제 진행 상황과 풀이 시간을 화면 상단에 표시합니다.
- **반응형 화면**: 데스크톱과 모바일 화면 폭에 맞춰 자동으로 배치됩니다.

## 실행 방법

1. `index.html` 파일을 웹 브라우저에서 엽니다. (별도 설치나 서버가 필요 없습니다.)
2. **CSV 파일 불러오기** 버튼으로 문제 파일을 선택하거나, 파일을 화면으로 끌어다 놓습니다. 파일 없이 살펴보려면 **샘플로 체험**을 누릅니다.
3. 초성·자모·한붓 중 원하는 방식을 골라 시작합니다.

### CSV 파일 형식

- 첫째 칸: 문제(설명) — 비워 두어도 됩니다.
- 둘째 칸: 정답
- 1행 머리글(예: `문제,정답`)은 자동으로 건너뜁니다.

예시 (`문제예시.csv`):

```csv
문제,정답
우리가 공부하는 곳,학교
빨갛고 둥근 과일,사과
친구와 뛰어노는 넓은 곳,운동장
,바나나
부모님을 공경하는 마음,효도
```

## 기술 스택

- HTML · CSS · 순수 JavaScript (단일 파일, 외부 프레임워크 없음)
- 한글 자모 분해 및 CSV 파싱 로직을 자체 구현 (BOM·따옴표·줄바꿈 처리 포함)
- 웹폰트: Pretendard, Instrument Sans
